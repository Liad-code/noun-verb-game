## MVP Scope

### Core Features
To deliver on the core value proposition quickly, the MVP will strictly include:
- **User Accounts:** Registration, login, and the ability to save trips to a dashboard.
- **Conversational Interface:** The core AI chat experience where users input their travel needs.
- **Contextual Memory:** Persistent context tracking within a session (AI remembers constraints like "toddler" or "window seat" throughout the dialogue).
- **Core Integrations:** Flight search via Kiwi and Skyscanner APIs; Hotel search via the Booking.com API.
- **Curated Delivery:** Results displayed as integrated cards inside the chat stream, featuring direct affiliate redirect links for booking.
- **User Profiles:** Basic preference saving (travel companions, class preference, budget range).

### Out of Scope for MVP
To prevent scope creep, the following are explicitly deferred to post-MVP (V2.0+) iterations:
- **Additional Verticals:** Car rentals, activities/experiences, and restaurant reservations.
- **Advanced Tools:** Price alerts, tracking, and real-time price comparison table views.
- **Social Features:** Group trip collaboration, sharing, and community features.
- **Platform Expansion:** Native mobile applications (iOS/Android).
- **Direct Processing:** Never in scope for direct booking/payment processing (the model remains 100% affiliate-based).
- **Localization/Retention:** Multi-language support and loyalty points integration.

### MVP Success Criteria
The MVP will be considered successful if it achieves:
- 500 registered users within 3 months.
- 100+ affiliate link clicks per month.
- 70%+ of started conversations reaching the curated results stage.
- 40%+ of completed chats resulting in at least 1 affiliate link click.

### Future Vision
If TRIPO achieves its MVP success criteria and secures market fit, the 2-3 year vision includes:
- **The Ultimate Trip OS:** Expanding to manage the entire trip lifecycle (flights, hotels, activities, restaurants, transfers) within a single conversation.
- **Proactive AI:** The system proactively suggests personalized trips based on user travel history and saved preferences.
- **B2B Expansion:** Offering a white-label B2B version of Tripo for use by travel agencies.
- **Community Hub:** Features for sharing itineraries, user reviews, and a centralized trip inspiration feed.
- **Premium Tier:** A subscription model offering advanced capabilities like price alerts, a concierge AI, and exclusive deals (e.g., lounge access).
